# Get Academy by ID

Returns a specific academy by its ID.


## 🔐 Authentication

❌ Not required

## 📥 Request Parameters

| Param | Type    | Required | Description           |
|-------|---------|----------|-----------------------|
| id    | integer | ✅ Yes   | The ID of the academy |

## 📤 Example Request

```http
GET /academy/1 HTTP/1.1

## 📤 Example Response

{
  "id": 1,
  "name": "Elite Academy",
  "location": "Algiers",
  "phone": "+213661234567"
}
```
## ✅ Status Codes

| code | meaning               |
| ---- | --------------------- |
| 200  | ok                    |
| 404  | Not found             |
| 500  | internal server error |
