# Get Academy by ID

Returns a specific academy by its ID.


## 🔐 Authentication

❌ Not required

## 📥 Request Parameters

| Param | Type    | Required | Description           |
|-------|---------|----------|-----------------------|
| id    | integer | ✅ Yes   | The ID of the academy |

## 📤 Example Request

```http
GET /academy/1 HTTP/1.1

## 📤 Example Response

{

  "name": "Elite Academy",
  "location": "Algiers",
