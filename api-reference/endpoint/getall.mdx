---
title: "Get all academies"
openapi: "GET /academy/all"
---

Returns a list of all academies in the system.

## 🛠️ Endpoint

POST /Academy/all

## 🔐 Authentication

❌ Not required

## 📥 Request Parameters

None

## 📤 Example Response

```json
[
  {
    "id": 1,
    "name": "Elite Academy",
    "location": "Algiers",
    "phone": "+213661234567"
  }
]
```
## ✅ Status Codes

| code | meaning               |
| ---- | --------------------- |
| 200  | ok                    |
| 500  | internal server error |
