# Create a New Academy

Creates a new academy in the system.

## 🛠️ Endpoint


## 🔐 Authentication

✅ Required (e.g., Bearer token)

## 📥 Request Body

Send a JSON object with the following fields:

| Field      | Type   | Required | Description              |
|------------|--------|----------|--------------------------|
| name       | string | ✅ Yes    | Name of the academy      |
| location   | string | ✅ Yes    | City where it is located |
| phone      | string | ✅ Yes    | Contact phone number     |

{
  "name": "Tech Genius Academy",
  "location": "Constantine",
  "phone": "+213550123456"
}
